# BCG Introduction to Data for Decision Makers

## 📊 Overview

This repository documents my completion of **BCG's Introduction to Data for Decision Makers** job simulation on Forage. It showcases the practical application of data analysis skills in a real-world business context, focusing on multi-channel ad campaign performance and data-driven decision making.

## 🎯 Project Summary

In this simulation, I completed a job simulation focused on advising a hypothetical client on multi-channel ad campaign performance. The project involved analyzing customer conversion trends using simulated data to assess campaign effectiveness across channels and audiences, ultimately synthesizing insights and delivering clear, data-backed recommendations tailored to the client's marketing goals.
